# TalaOnline - سیستم مدیریت قیمت طلا و ارز

این پروژه یک پلتفرم جامع برای مشاهده و مقایسه قیمت‌های لحظه‌ای طلا، سکه و ارز است که داده‌های خود را از وب‌سرویس نوسان دریافت می‌کند.

## ویژگی‌ها
- **پنل مدیریت اختصاصی:** مدیریت آیتم‌ها، پلتفرم‌ها و تنظیمات سیستم.
- **اصلاح دستی قیمت:** امکان تغییر قیمت‌ها به صورت دستی بدون نیاز به تغییر در API.
- **نصاب خودکار:** راه‌اندازی آسان دیتابیس و تنظیمات اولیه.
- **معماری جداگانه:** تفکیک کامل فرانت‌اِند (استاتیک) از بک‌اِند (PHP API).

## نحوه نصب
1. فایل‌های پروژه را در هاست آپلود کنید.
2. یک دیتابیس MySQL بسازید.
3. به مسیر `mysite.com/installer.php` بروید و مراحل را دنبال کنید.
4. پس از اتمام نصب، برای امنیت بیشتر فایل `installer.php` را حذف کنید.

## ساختار پوشه‌ها
- `/admin`: پنل مدیریت حرفه‌ای.
- `/api`: اندپوینت‌های API برای فرانت‌اِند.
- `/includes`: توابع هسته و ارتباط با دیتابیس.
- `/css`, `/js`, `/images`: فایل‌های استاتیک فرانت‌اِند.
